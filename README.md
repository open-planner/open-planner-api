# OpenPlanner API

## Requisitos

* JDK11
* PostgreSQL 10
* JUnit 5

## Recursos

* Autenticação com OAuth2 e JWT
* Usuário
  * Listagem
  * Cadastro (*com confirmação por e-mail*)
  * Edição de Informações Pessoais
  * Edição de Senha
  * Alteração de Status
  * Recuperação de Login
  * Recuperação de Senha
